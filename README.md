## 📋 О проекте

**DiscountPRO** — это веб-приложение для создания и управления дисконтными программами. Система позволяет:

- ✅ Создавать и управлять дисконтными акциями
- ✅ Категоризировать акции по типам
- ✅ Добавлять теги для удобного поиска
- ✅ Загружать изображения для акций
- ✅ Фильтровать акции по категориям и тегам
- ✅ Интеграция с Яндекс.Картами
- ✅ ИИ-ассистент на базе YandexGPT

---

## 🚀 Функциональность

| Функция | Описание |
|---------|----------|
| 📝 **Управление акциями** | CRUD операции для дисконтных программ |
| 🏷️ **Категории и теги** | Гибкая система фильтрации |
| 🖼️ **Загрузка изображений** | Поддержка фото для каждой акции |
| 🔍 **Поиск и фильтрация** | Быстрый поиск по категориям и тегам |
| 🗺️ **Яндекс.Карты** | Отображение офиса на карте |
| 🤖 **ИИ-ассистент** | Чат-бот на базе YandexGPT |
| 👑 **Админ-панель** | Удобное управление через Django Admin |
